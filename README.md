# 💼 Personal Portfolio Website

A modern, responsive, and elegant **Personal Portfolio Website** built using **HTML5, CSS3, and JavaScript**. This project showcases a developer's profile, technical skills, projects, and contact information through a clean and professional user interface.

The website is fully responsive and optimized for desktops, tablets, and mobile devices, making it an excellent portfolio for students, developers, and freelancers.

---

# 🌟 Features

* 🏠 Beautiful Landing (Hero) Section
* 👨‍💻 About Me Section
* 💡 Skills Showcase
* 📂 Projects Gallery
* 📞 Contact Information
* 📱 Fully Responsive Layout
* 🎨 Clean & Modern UI Design
* ✨ Smooth Scrolling Navigation
